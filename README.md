# ML Projects

A curated portfolio of machine learning work spanning **reinforcement learning, deep learning, applied ML, and from-scratch algorithm implementations**. Each entry below links to its own repository containing code, notebooks, and writeups.

---

## Projects

### Algorithms From Scratch
- **[ML Algorithms From Scratch](https://github.com/nossyd/ML-projects/tree/main/ml-algorithms-from-scratch)** — Classical ML algorithms (e.g., Learning Vector Quantization) implemented in NumPy from first principles to solidify the underlying math and learning dynamics.

### Deep Learning
- **[PyTorch Fundamentals](https://github.com/nossyd/ML-projects/tree/main/pytorch-fundamentals)** — Distributed multi-GPU neural network training with PyTorch's `DistributedDataParallel`, covering process groups, data sharding, and gradient synchronization.
- **[JAX Fundamentals](https://github.com/nossyd/ML-projects/tree/main/jax-fundamentals)** — A Titanic survival classifier built in JAX to develop intuition for `jit`, `vmap`, `grad`, and functional-style model code.

### Applied ML
- **[Predicting Click-Through Rate](https://github.com/nossyd/ML-projects/tree/main/predicting-CTR-with-ML-in-python)** — End-to-end CTR prediction pipeline: feature engineering, Random Forest modeling, and hyperparameter tuning via grid-search cross-validation.
- **[Customer Wait Time Prediction](https://github.com/nossyd/ML-projects/tree/main/predict-customer-wait-time)** — Regression problem framed around real-world operational data, including weather-data EDA, cleaning, and feature construction.

### Reinforcement Learning
- **[Monte Carlo Policy Evaluation](https://github.com/nossyd/ML-projects/blob/main/ml-algorithms-from-scratch/Reinforcement-Learning/monte-carlo-policy-evaluation.ipynb)** — Model-free policy evaluation using Monte Carlo methods in tabular environments. Explores the bias/variance trade-offs of MC vs. bootstrapped estimators.
